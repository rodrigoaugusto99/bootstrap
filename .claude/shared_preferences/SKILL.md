# SharedPreferences

Use para: onboarding, flags de primeira execução, preferências simples de UI. Não use para entidades de domínio ou dados críticos. Sempre use enums para keys — nunca strings soltas. → [ex_shared_preferences.dart](references/ex_shared_preferences.dart)
